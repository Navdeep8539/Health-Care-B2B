# 🏥 Health Care Dashboard (B2B)

A modern and responsive Healthcare Dashboard built using **React, TypeScript, Tailwind CSS, and Vite**.  
This application is designed to manage patients, monitor data, and provide an intuitive UI for healthcare management systems.

---

## 🚀 Features

- 🔐 Authentication (Protected Routes)
- 👨‍⚕️ Patient Management (Add, Edit, Delete)
- 📊 Dashboard with Data Visualization
- 🔍 Search & Filter Patients
- 📱 Fully Responsive UI
- ⚡ Fast performance with Vite
- 🧩 Reusable UI Components (ShadCN + Radix UI)

---

## 🛠️ Tech Stack

**Frontend:**
- React.js (TypeScript)
- Vite
- Tailwind CSS
- ShadCN UI + Radix UI

**State & Data:**
- React Query (TanStack)
- Zustand

**Backend / Services:**
- Supabase / Firebase

**Other Tools:**
- React Hook Form + Zod (Validation)
- Recharts (Charts)
- ESLint + Prettier
